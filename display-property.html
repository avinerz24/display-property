DISPLAY PROPERTY REFERENCE

Value Name Description Example
Block Starts on a new line and takes up the whole screen width. Think <div>, <section>, or <p> "This is a
block
element."
Inline Does not start on a new line, takes up the remaining/available screen width, and height/width cannot be adjusted. Think <strong>, <a>, or <img> "This is an
inline
element."
Inline-Block By presentation, this is Inline. Elements can appear on the same horizontal line as other elements but the width/height can be adjusted unlike Inline, so this is like a combination of Block and Inline-Block. "This is an
inline-block
element."
None Hides the element from the website and it will not be shown visually. "This is a
none
element." (The word "none" is not displayed.)
